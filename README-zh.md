## 如何使用
1. 配置当前项目和目标项目
2. 运行 `script` 脚本

## 文件说明
- **index.js**: 项目的主文件，包含主要的业务逻辑。
- **promt.md**: 项目的提示文件，包含一些提示信息。
- **README.md**: 项目的说明文件，包含项目的使用说明和结构。

## 运行项目
在终端中运行以下命令来启动项目：

```sh
node index.js
```
